Galerie My Config

My Config est un site qui permet de connaître la configuration de son ordinateur. Il est très utile pour les personnes qui ne connaissent pas leur configuration ou qui veulent acheter un ordinateur et qui veulent connaître la configuration de celui-ci. Projet réaliser en groupe de 4 personnes. Il a pour but de permettre à l'utilisateur de connaître la configuration de son ordinateur.
